1. Requirements — Kya Chahiye

Software Requirements

  • Node.js v18+ — JavaScript runtime (check: node -v)
  • Python 3.10+ — PC Bridge ke liye (check: python --version)
  • npm — Package manager (Node ke saath aata hai)
  • pip — Python package manager
  • Modern Browser — Chrome / Edge / Firefox (koi bhi)

API Keys — Koi EK chahiye

  • Groq — Free, sabse fast ⚡ → console.groq.com
  • Google Gemini — Free tier available → aistudio.google.com
  • Mistral — Free tier → console.mistral.ai
  • Cerebras — Free, ultra fast → cloud.cerebras.ai
💡 Beginner? Groq use karo — free hai, fast hai, aur sign-up mein sirf 1 minute lagega. Google account se directly login ho jaata hai.

2. Setup — Install Kaise Karein

🚀 One-Click Launcher & Auto-Install: Sirf start.bat file double-click karo — yeh command pehli baar chalne par sabhi dependencies (Python libraries jaise psutil, edge-tts, pyautogui, pyttsx3 aur frontend Node modules) ko apne aap download aur install kar legi! Iske baad web UI aur backend dono server ek saath start ho jayenge.
1

Repository clone karo ya download karo

Terminal
git clone https://github.com/SudhirDevOps1/Ai-assistant-pika.git cd Ai-assistant-pika
2

Saari dependencies install karo

npm
npm install # Next.js, React, Tailwind, etc. download hoga — 2-5 min lagega
3

Development server start karo

npm
npm run dev # Ya seedha start.bat double-click karo (Windows)
4

Browser mein open karo

URL
http://localhost:3000

Boom! Pika AI Assistant open ho jaayega

📱 Phone se bhi access karo! Same WiFi pe hona chahiye — phone browser mein http://YOUR_PC_IP:3000 type karo. Apna PC IP check karo: ipconfig (Windows) ya ifconfig (Mac/Linux) command se.

3. PC Bridge — Python Server

⚠️ Zaroori: PC Bridge sirf PC Control features ke liye chahiye (volume, apps, shutdown etc.). Agar sirf AI chat use karna hai toh iske bina bhi chalega!

PC Bridge kya hai?

Yeh ek Python WebSocket server hai jo tumhare PC pe run hota hai. Web UI isse connect hokar commands bhejta hai — "volume up karo", "Chrome kholo", "PC shutdown karo" etc. Ya phir start.bat se sab ek saath start ho jaata hai.

6b. Vyron Dashboard & Project Architecture Guide

Vyron A.I. 3-Column Grid Interface Layout

Naya futuristic matte-black dashboard 3 columns mein structured hai:

  • Left Column (Diagnostics & Stack):
    • Network Telemetry: Connection speed and avg latency (10ms).
    • Core Metrics: CPU, RAM, Temp, aur OS ke real-time status details.
    • Webcam Feed: Purple toggle switch "ON" karne par browser camera stream start ho jata hai.
    • Built With the Best: Technology specs details (Gemini Flash, Python, pyautogui, Whisper, Next.js, PyInstaller, SQLite 3).
  • Center Column (Neural Visualizer & Call Controls):
    • SVG Orbiting Rings: Rotating gyroscopes jo system status (Listening, Speaking, Thinking, Idle) ke sath react karte hain.
    • Voice Control Buttons: Recording active mic indicator (Mic toggle), central voice call trigger, aur manual chat toggler.
  • Right Column (Transcript Logs & Manual Typing Chat):
    • 📜 Transcript logs: Chat history displays in transparent bordered bubble styling.
    • Manual typing input: Agar aapko bolna nahi hai, toh keyboard button dabake chat logs pane open karein aur seedhe type karke PC commands chalwayein!

Smart Offline/Online Voice Engine & PiP Floating Window

  • Zero-Delay TTS Fallback: Edge-TTS online check karne ke liye background mein ek fast DNS query (0.6s timeout) chalti hai. Agar aap completely offline hain, toh server 4-second Bing timeout delays ke bina instantly offline woyce engine (Pyttsx3) pe switch kar leta hai.
  • PiP Floating Window: Isko dekhne ke liye settings panel (⚙️) pe jayein aur "Picture-in-Picture" Toggle Switch ON karein. Yeh widget screen pe humesha top pe rahega aur live action logs (jaise 📄 Creating file...) display karega.

Project Files Audit (Kaam Ka vs Bekar Ka Details)

Pro professional audit ke mutabik kaun si file kis kaam aati hai:

11. Troubleshooting — Fix Problems

"No API key configured"

Fix: Settings → API key daalo → Save Settings. Key console.groq.com se milegi.

PC Control kaam nahi kar raha

  • Python bridge chal raha hai? (python pc_bridge.py)
  • Settings mein ws://localhost:8765 daala?
  • Save Settings button dabaya?
  • Firewall allow kiya? (Windows Defender → Allow an app)

npm run dev error aa raha

Fix Commands
rmdir /s /q node_modules del package-lock.json npm install npm run dev

Port 3000 already in use

Kill Port
npx kill-port 3000 npm run dev

Python bridge Unicode error

Terminal
chcp 65001 python pc_bridge.py

Volume/Media controls kaam nahi

Fix: pip install pyautogui — yeh package zaroori hai volume, media, aur keyboard controls ke liye.

Voice recognition kaam nahi kar raha

Fix:

  • Browser ko microphone permission do (lock icon → Allow)
  • Chrome ya Edge use karo (Firefox mein limited support)
  • HTTPS ya localhost pe hona chahiye
