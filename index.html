System Architecture

Yeh Kaam Kaise Karta Hai?

Pika AI ek hybrid architecture use karta hai jahan Web UI aur Local Python Server mil kar kaam karte hain:

┌─────────────────┐ WebSocket ┌───────────────────┐ OS API ┌────────────┐ │ Next.js UI │ ◄────────────────────► │ Python Bridge │ ◄──────────────► │ Windows │ │ (Browser/React)│ (ws://localhost) │ (pc_bridge.py) │ (pyautogui, etc) │ System │ └─────────────────┘ └───────────────────┘ └────────────┘
  • Frontend: Next.js aur React se bana beautiful UI jahan aap chat aur controls dekhte ho.
  • Backend Bridge: Python server jo browser se commands leta hai aur PC ke hardware/OS ko control karta hai.
  • AI Brain: Groq, Gemini, Mistral ya Cerebras ke APIs jo natural language ko samajhte hain.

Installation Setup

1

Clone & Navigate

Terminal mein project folder open karein.

cd E:\tmp\0606\AI\AI
2

Install Frontend Dependencies

Next.js aur React packages install karein.

npm install
3

Start Development Server

UI ko localhost par run karein.

npm run dev

Browser mein http://localhost:3000 open karein.

PC Bridge (Python Server)

⚠️ Note: Yeh step sirf tab zaroori hai agar aapko PC Control (Volume, Apps, Shutdown) karna hai. Sirf Chat ke liye iski zaroorat nahi hai.
1

Install Python Dependencies

Naye terminal mein `pc-bridge` folder mein jaayein:

cd E:\tmp\0606\AI\AI\pc-bridge
pip install -r requirements.txt
2

Start WebSocket Server

python pc_bridge.py

Server ws://localhost:8765 par start ho jayega.

⚙️

Configuration

🔗 Connecting the Dots

Web UI (localhost:3000) mein Settings panel open karein aur yeh cheezein set karein:

  • AI Provider: Groq select karein (Fastest).
  • API Key: Apni Groq/Gemini API key paste karein.
  • PC Bridge URL: ws://localhost:8765 daalein.
  • Voice Settings: Wake word ("Hey Pika") aur TTS voice set karein.

✅ Save Settings par click karein. Aapka assistant ready hai!

Security & Privacy

  • Local-First: PC Bridge sirf `localhost` par run hota hai. Koi bahari access nahi.
  • API Keys: Aapki keys sirf aapke browser (LocalStorage) mein save hoti hain, kisi server par nahi.
  • Open Source: Pura code transparent hai, aap khud verify kar sakte hain.

Troubleshooting

❌ "No API Key" Error

Settings mein jaakar apne selected provider ki valid API key daalein aur Save karein.

❌ PC Control Not Working

Check karein ki `python pc_bridge.py` chal raha hai aur Settings mein WebSocket URL `ws://localhost:8765` hai.

❌ Port 3000 in Use

npx kill-port 3000

❌ Python Unicode Error

Windows CMD mein pehle chcp 65001 run karein, phir script start karein.
