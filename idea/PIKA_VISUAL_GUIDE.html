🤖 Pika Kya Hai? What is Pika?

Pika ek local PC voice-controlled AI assistant hai jo aapke computer ko Hindi aur English dono mein control karne deta hai. Ye completely offline-first hai — aapka data kisi server ko nahi jaata.

🎤

Voice Control

Offline STT via Vosk, Hindi + English support. "Chrome kholo", "volume up karo" — bas bolo!

🖥️

PC Automation

30+ commands: apps open/close, volume, media, files, clipboard, screen, processes, network.

🧠

AI Chat (7 Providers)

Groq, Gemini, Mistral, Cerebras, OpenRouter, Z.ai, DeepSeek — free API keys se chalta hai.

🚀 Setup Guide — Pika Kaise Chalayein

1

Prerequisites Install Karo

Ye cheezein chahiye:

  • Python 3.10+ — python.org se download karo. Install karte time "Add to PATH" check karna mat bhoolna!
  • Node.js 18+ — nodejs.org se LTS version install karo.
  • Git (optional) — git-scm.com
# Verify installations:
python --version    # Python 3.10.x+
node --version      # v18.x.x+
npm --version       # 9.x.x+
2

Project Download / Clone Karo

# Folder extract karo ya clone karo
cd pika-ai-assistant
3

One-Click Start (Recommended)

Windows: start.bat file double-click karo. Ye sab automatically karega:

  1. Python dependencies install karega
  2. Node.js dependencies install karega
  3. Required folders banayega (models, data, screenshots, etc.)
  4. Python backend start karega (ws://localhost:8765)
  5. Frontend start karega (http://localhost:3000)
  6. Browser automatically open hoga

Linux/Mac: chmod +x start.sh && ./start.sh

4

Manual Start (agar auto-start fail ho)

Terminal 1 — Backend

cd pika-ai-assistant
pip install -r requirements.txt
python pc_bridge.py

# Output: 🔗 Pika AI Bridge running on ws://localhost:8765

Terminal 2 — Frontend

cd pika-ai-assistant/frontend
npm install
npm run dev

# Output: ➜ Local: http://localhost:3000
5

API Keys Setup (Optional but Recommended)

AI chat ke liye free API keys chahiye. Bina keys ke bhi Pika chalega (PC commands kaam karenge, sirf AI chat nahi hoga).

.env.example ko copy karke .env banao aur keys daalo:

# Copy the template
cp .env.example .env

# Edit .env and add your keys (sab optional hain!)
GROQ_API_KEY=gsk_your_key_here      # https://console.groq.com (FREE, recommended)
GEMINI_API_KEY=your_key_here         # https://aistudio.google.com (FREE)
DEEPSEEK_API_KEY=your_key_here       # https://platform.deepseek.com (FREE)

Pika auto-fallback use karta hai — agar ek provider fail ho toh doosra try karta hai. Ek bhi key daalna kaafi hai!

6

Browser Mein Khulo

Open karo: http://localhost:3000

Pika ka UI load hoga — connection status top-right mein dikhega. 🟢 Connected dikhna chahiye.

🔧 Troubleshooting — Common Problems

🔴 "WebSocket not connected" error

Cause: Python backend (pc_bridge.py) nahi chal raha.

Fix: Dusre terminal mein python pc_bridge.py run karo. 🔗 Pika AI Bridge running on ws://localhost:8765 message dikhna chahiye.

🟡 "Some Python packages failed to install"

Cause: Koi optional library install nahi hui (e.g., pytesseract needs tesseract-ocr system package).

Fix: Pika bina iske bhi chalega — sirf us feature kaam nahi karega. Error mein bataya hoga kaun si package missing hai.

🟡 "AI not responding" / "No API key"

Cause: Koi bhi LLM provider ka API key configure nahi hai.

Fix: .env file mein kam se kam ek key daalo (Groq recommended — free hai). Settings → AI Provider mein bhi daal sakte ho.

🔵 Voice not working

Cause: Vosk models download nahi huye, ya microphone permission nahi di.

Fix: Pehli baar start hone pe models auto-download honge (~50MB each). Browser mein mic permission allow karo.

🔵 Port 3000 or 8765 already in use

Fix: netstat -ano | findstr :3000 (Windows) ya lsof -i :3000 (Mac/Linux) — us process ko kill karo. Ya different ports use karo settings mein.
