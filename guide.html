3. PC Bridge — Python Server Setup

⚠️ Zaroori: PC Bridge SIRF PC Control features ke liye chahiye (volume, apps, shutdown, etc.). Agar sirf AI chat use karna hai toh iske bina bhi kaam chalega!

🐍 PC Bridge kya hai?

Yeh ek Python WebSocket server hai jo tumhare PC pe run hota hai. Web UI isse connect hota hai aur commands bhejta hai jaise "volume up karo", "Chrome kholo", "PC shutdown karo" etc.


Flow:

Web UI (Browser) ──WebSocket──► Python Bridge (pc_bridge.py) ──► PC Actions localhost:3000 localhost:8765 Volume, Apps, etc.
1

Nayi Terminal/CMD kholo (Web server wali band mat karna!)

Yeh alag terminal honi chahiye — ek mein web server chalega, ek mein Python bridge

2

PC Bridge folder mein jaao

cd E:\tmp\0606\AI\AI\pc-bridge
3

Python dependencies install karo (sirf pehli baar)

pip install -r requirements.txt

Yeh install karega: websockets, psutil, pyautogui, pyperclip, pygetwindow, Pillow

4

Python Bridge start karo

python pc_bridge.py

Output aayega:

+--------------------------------------------------------------+ | PC Bridge -- WebSocket Server | | Pika AI Assistant Desktop Assistant | | | | Server running on: ws://0.0.0.0:8765 | | Local access: ws://localhost:8765 | +--------------------------------------------------------------+ Your LAN IP: ws://192.168.1.59:8765
5

Web UI mein Settings karo

Browser mein http://localhost:3000 kholo → Sidebar mein ⚙️ Settings click karo → Neeche scroll karo → "PC Bridge (WebSocket)" section mein:

ws://localhost:8765

Yeh URL paste karo aur Save Settings button dabao!

✅ Done! Ab sidebar mein 💻 PC Control tab pe jaao — connected dikhega aur saare controls kaam karenge!
⚙️

4. Settings — Configure Karna

⚡ Provider & Model

Kaunsa AI provider use karna hai choose karo. 4 options hain:

  • Groq (Default) — Sabse fast, free, recommended for beginners
  • Gemini — Google ka AI, bahut smart
  • Mistral — European AI, coding mein strong
  • Cerebras — Ultra fast inference

Provider select karne ke baad uska model choose karo (har provider ke multiple models hain)

🔑 API Keys

Jo bhi provider use karna hai uski API key daalni hogi. Example Groq ke liye:

1

console.groq.com pe jaao → Sign up karo (Google se) → API Keys section mein "Create API Key" → Key copy karo

2

Web UI → Settings → Groq API Key field mein paste karo → Save Settings

🔊 Voice Settings

  • TTS Voice — Text-to-Speech ki voice (default: en-US-Neural2-A)
  • Wake Word — Voice activate karne ka word (default: "Hey Assistant")
  • Conversation History Limit — Kitni purani messages yaad rakhe (default: 50)
  • Auto-fallback — Agar ek provider fail ho toh automatically doosre pe switch kare

🔌 PC Bridge (WebSocket)

PC Control features ke liye Python bridge ka URL daalo:

# Agar same PC pe hai:
ws://localhost:8765

# Agar doosre PC pe hai (uska IP daalo):
ws://192.168.1.100:8765

11. Troubleshooting — Problems Fix Karna

❌ "No API key configured" error

Fix: Settings → apne provider ki API key daalo → Save Settings

❌ PC Control kaam nahi kar raha

Fix: Check karo ki:

  • Python bridge chal raha hai (python pc_bridge.py)
  • Settings mein ws://localhost:8765 daala hai
  • Save Settings button dabaya hai
  • Firewall block nahi kar raha (Windows Defender mein allow karo)

❌ npm run dev error

Fix:

# node_modules delete karke reinstall karo:
rmdir /s /q node_modules
del package-lock.json
npm install
npm run dev

❌ Port 3000 already in use

Fix:

# Pehle wala process kill karo:
npx kill-port 3000
npm run dev

❌ Python bridge crash / Unicode error

Fix: Terminal mein UTF-8 set karo:

chcp 65001
python pc_bridge.py

❌ Volume/Media controls kaam nahi kar rahe

Fix: pip install pyautogui — yeh zaroori hai volume/media/keyboard controls ke liye
